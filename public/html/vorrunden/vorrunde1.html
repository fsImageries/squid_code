Variablen

Variablen
sind benannte Werte und können jede Art von JavaScript-Werten speichern.

Hier siehst du wie man die Variablen deklarieren kann:

EXAMPLE
let x = 100;
const y = 50;

Und hier was im oberen Beispiel passiert:

  • let ist ein Stichwort zum deklarieren der Variable.
  • x ist der Name von dieser Variable.
  • = ist der Operator und sagt JS, dass als nächstes ein Wert folgt.
  • 100 ist der Wert der gespeichert werden soll.

Variablen benutzen

Nachdem du eine Variable deklariest hat, kannst du dich über den gesamten Code auf den Namen beziehen.

EXAMPLE
let x = 100;
let y = x + 102
y;
OUTPUT
202

Variablen neu zuweisen

Du kannst ein bereites existierende Variable einen neuen Wert geben.
Achtung! Das geht aber nur mit

let

. Die

const

-Deklaration erstellt eine Konstante. Der Wert einer Konstanten kann nicht verändert werden durch Zuweisung oder Neudeklaration.

EXAMPLE
let light = "red";
light = "green";
light;
OUTPUT
"green"

Variablen benennen

Das benennen von Variablen weist sich als recht flexibel, solang du folgende Regeln folgst:

  • Beginne mit einem Buchstaben, Underscore _, oder Dollar-Zeichen $.
  • Nach dem ersten Buchstaben kannst du Zahlen benutzen, als auch Buchstaben, Underscore _, oder Dollar-Zeichen $.
  • Benutze keine Stichwörte, die von JavaScript bereits festgelegt worden sind.

Hier siehst du ein paar Bespiele von gültigen Variablennamen:

EXAMPLE
let squidGame = "2021";
let _Guard = "triangle";
let NumberOfPlayers = 456;
let §PricePool = 45.6 "billion won";

Und hier sind Bespiele von ungültigen Variablennamen:

EXAMPLE
let class = "FrontMan";
let 2stop2go = "triangle";
let function = false;
let winchance% = "zero";

Variablennamen sind Fallsensetiv, d.h. myvar, myVar, MyVar sind unterschiedliche Variablen.
